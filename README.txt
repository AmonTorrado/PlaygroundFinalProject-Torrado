Descripción del funcionamiento de la página:

Al ingresar a la dirección incial luego de ejecutar runserver, se puede crear usuario o explorar la página como visitante. Esta es una página de compra de insumos de computación, para
poder comprar elementos en la página se tiene que estar registrado. En caso de que la persona se registre como un superusuario, tiene ventajas de poder agregar o eliminar productos a la venta.
Además puede cambiar los parámetros de los productos. Un usuario ordinario no tiene estos privilegios, solo pueden limitarse a ver y comprar productos.
El menú de opciones a utilizar se mostrará distinto para el dueño del negocio que para el comprador.
