Descripción de la tarea:

Al ingresar a la dirección incial luego de ejecutar runserver, se puede elegir entre las 4 clases incluidas en la tarea. Se puede elegir cualquiera de ellas e ingresar
la información a la base de datos.

En el menú inicio, se encuentra el formulario para buscar algo en la base de datos.

En /admin se pueden observar los usuarios creados y verificar que los datos ingresados fueron guardados en la base de datos.
